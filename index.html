Projetos

  • Practice Hanzi

    Permite que a pessoa usuária use o mouse para praticar a escrita de caracteres chineses. A pessoa usuária também pode salvar o seu desenho como PNG. Atualmente adaptando para o uso em celular com resposta ao toque.

  • Timer Bozo

    Mostra os dias, horas, minutos e segundos até o fim do governo Bolsonaro. À medida que o tempo vai diminuindo, o fundo vai ficando mais próximo do vermelho, até que, quando a contagem chegar a 0, a mensagem de texto fica invisível.

  • Beba Água

    O site pergunta à pessoa usuária quantos copos de água já bebeu no dia de hoje. Assim que ela responde, a animação mostra a quantidade de água que falta para chegar ao ideal diário de consumo. Site trilíngue (en-pt-zh).

  • Site pessoal

    Modelo de website pessoal ou portfolio, contendo informações de contato, projetos, currículo, etc. Ao ativar o menu de navegação, a página que está sendo mostrada gira 20 graus.

  • Pong!

    Jogo clássico recriado usando JavaScript e HTML5 Canvas API.

  • Notes App

    Web-app para escrever e tomar notas. O usuário pode escrever suas notas usando Markdown, elas serão convertidas automaticamente para texto formatado. As notas são salvas no localStorage do navegador.
